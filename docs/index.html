Glossolalia is a domain-specific language for the reproducible generation of words in constructed (made-up) languages. Design your conlang's sounds, syllable structure, and word construction rules in a simple declarative language, then run the file to get a list of generated words! Run the examples below after installing to see what Glossolalia can do, or learn more from the description of the language.
#lang glossolalia Sounds ------ @onset = m, n, p, k, ', h, w, l @vowel = a, e, i, o, u @long = ā, ē, ī, ō, ū @dipth = iu, ou, au, eu, ou, ei, ai, ao, ae, ōu, ēi, āu, āi, āo, āe Syllables --------- $one = @onset @vowel = 25% $two = @vowel $three = @onset @long = 25% $four = @long $five = @onset @dipth = 25% $six = @dipth Rules ----- w never-followed-by ū w never-followed-by u Generate -------- Seed = 12345 Count = 100 Longest = 5 #lang glossolalia Sounds ------ @onset = p, t, k, v, r, g @short = a, e, i, o, u @long = aa, ee, ii, oo, uu, ie, ia, io, iu, ei, ea, eo, eu, ai, ae, ao, au, oi, oe, oa, ou, ui, ue, ua, uo Syllables --------- $one = @short $two = @long $three = @onset @short = 50% $four = @onset @long = 30% Rules ----- @long never-starts-word Generate -------- Seed = 9001 Count = 100 Longest = 6 #lang glossolalia Sounds ------ Syllables --------- Rules ----- Generate --------
