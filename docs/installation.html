There are two conventional methods for getting Glossolalia on to your system: as a package in an existing installation of the Racket programming language, or as a standalone executable.

1. As a #lang in Racket

  1. Make sure you have a relatively recent version of Racket installed.
  2. Open a terminal and type raco pkg install glossolalia
  3. Run your Glossolalia files using the racket <myconlang.rkt> command or open them in Dr. Racket and hit the play button.

This option is the recommended one at the moment, as it is cross-platform and gives you all the good things about Racket, including the Dr. Racket IDE. However, it also means you have to install Racket, which could be annoying if all you want is this tool.

2. As a standalone application (currently Windows only)

  1. Download the latest stable release and unzip it into the desired location.
  2. Run the included executable glossolalia.exe from the command line on your source file.

Unfortunately, this option isn't as nice as it could be just yet, only supporting Windows and with no installer, which means you'll have to add it to the path yourself if you want to use it from anywhere.
